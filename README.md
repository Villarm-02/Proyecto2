# Strategic Game - Juego de Estrategia por Consola

Proyecto desarrollado en C# basado en un sistema de combate estilo ajedrez, utilizando conceptos de Programación Orientada a Objetos (POO) como encapsulamiento, abstracción, polimorfismo e interfaces.
El juego funciona completamente en consola e incluye un sistema de login, tablero visual, historial de movimientos, sistema de puntuación y validación de reglas para cada pieza.
